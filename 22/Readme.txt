Added the following features to the ToDo List software:
1. Tick tasks (done in database and UPDATE database)
2. Deleting tasks (DELETE in the database)
3. Sorting the display in the form of done tasks at the bottom and undone tasks at the top
4. Adding the Priority column to the tasks table, to specify the important tasks from the normal tasks. Display this theme in ui with different colors.
5. In the add new task section, define a field to receive the date and time from the user.
6. If we click on a task, the details of its description and time will be displayed.
7. Create executable exe file from the software using pyinstaller
